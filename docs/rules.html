Performance

  • No AFK farming; farms must need active involvement but Low-health mob farms are allowed. ‚ùì

    • AFK farms require significant server resources to maintain optimal gameplay. To ensure a smooth experience for all players, we discourage the use of AFK farms.

  • No massive mob farms. ‚ùì

    • Large farms can cause performance issues for the server.

  • Keep hopper/redstone contraptions small. ‚ùì

    • Large contraptions can create lag, affecting the experience for all players.

  • No significant performance issues. ‚ùì

    • Maintaining performance ensures smooth gameplay for everyone.
